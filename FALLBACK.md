# 0건 재검색 전략

검색 결과가 0개면 아래 순서대로 조건을 완화해서 재검색:

| 단계 | 완화 내용 |
|---|---|
| 1 | ui_pattern 제거 → app + category + month만으로 재검색 |
| 2 | app 제거 → ui_pattern + category만으로 재검색 |
| 3 | month를 Drive에 수집된 가장 최근 월로 교체 후 재검색 |
| 4 | 모든 완화 후에도 0개 → 되묻기 |

단계 3 결과 있을 때 안내 문구:
{입력월}에 수집된 화면이 없어서 가장 최근에 수집된 {최근월} 기준으로 찾은 결과예요.

단계 3 결과 없을 때:
- 그 다음 최근 월 순서로 순차 시도
- 모든 월 시도 후에도 0건이면 되묻기

단계 1 조건이 모두 None인 경우:
- ui_pattern 제거 시 조건이 전부 None이 되면 재검색하지 않고 바로 되묻기

재검색 시 사용자에게 조용히 알림:
조건을 조금 넓혀서 다시 검색할게요.

---

# 되묻기 형식

모든 조건 완화 후에도 0건이면:
검색 결과가 없어요.
조건을 바꿔볼까요?
- {ui_pattern} 화면을 다른 앱 이름으로 검색 (ui_pattern 없으면 "다른 앱 이름으로 검색")
- ui_pattern 변경 (예: 로그인/회원가입, 추천/메인, 리스트)
- 수집월 지정 (예: 4월, 2026.03)

되묻기 문구 생성 규칙은 CONFIG.md의 한국어 변환 규칙 참조.

앱 이름이 불분명할 때:
어떤 앱 스크린샷을 찾을까요?
앱 이름을 알려주시면 더 정확하게 검색할게요.

