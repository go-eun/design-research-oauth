# 사용자 상호작용 규칙

## 대량 결과 처리
- 검색 결과가 30개를 초과하면 다운로드 전에 반드시 확인 요청
- 확인 문구: "검색 결과가 N개예요. 모두 다운로드할까요? (y/n)"
- y: 전체 다운로드 진행
- n: 이전 조건을 한 줄로 보여주고 새 조건 입력 요청 (기존 조건 초기화)
  이전 조건: {app} / {ui_pattern} / {month}
  새 조건을 입력해주세요. (초기화됩니다)
- 재검색 후 30개 초과면 동일한 y/n 질문 반복
- 재검색 후 30개 이하면 바로 다운로드

## 입력 가이드
- 찾고 싶은 서비스가 있다면 공식 서비스명을 입력해주세요.
  '앱', '서비스', '어플'은 이름 뒤에 붙여도 자동으로 인식돼요.
  예: 강남언니 / 헬스앱 / 카카오뱅크 서비스

- '이번달', '지난달'로 입력하면 자동으로 날짜를 계산해서 해당 월에 수집된 화면을 찾아줘요.
  예: 이번달 / 지난달 / 4월 / 2026.04

- 아래 조건을 조합하면 더 명확한 결과를 찾을 수 있어요.
  서비스명 · 화면 특징 · 수집 시점
  예: 강남언니 로그인 이번달 / 헬스앱 온보딩 2026.03

## 붙여쓰기 감지
- 사용자 입력에서 앱 이름, ui_pattern 한국어, 카테고리 키워드가 붙어서 입력된 경우 붙여쓰기로 감지
- 감지 시 검색하지 말고 아래 형식으로 안내:
  띄어쓰기를 지켜서 입력해 주세요.
  '{감지된 키워드1} {감지된 키워드2}'로 다시 입력해 주시면 검색할게요.
- 사전에 없는 단어 조합은 감지하지 못할 수 있음 → 0건 재검색 전략으로 처리
- 감지 후 사용자 재입력 대기
