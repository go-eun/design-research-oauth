# 검색 조건 추출 규칙

## app
- 앱 이름이 언급되면 추출
- 따옴표로 감싼 경우 최우선 추출
- 앱 이름이 불분명하면 추출하지 않고 ui_pattern만으로 검색

## ui_pattern
- 사용자 입력 키워드를 변환 없이 그대로 --ui_pattern에 전달
- 키워드 → 정식 패턴명 변환은 search_drive.py의 expand_ui_pattern()이 전담
- 패턴 목록은 CONFIG.md 참조

## category
- 한국어/영문 모두 인식
- 유의어 매핑은 search_drive.py의 CATEGORY_SYNONYMS 사전이 처리
- "~앱", "~서비스", "~어플" 접미어 자동 제거 후 매칭
  예: "운동앱" → "운동" → health-fitness
- CONFIG.md의 14개 카테고리 참조

## month
- yyyy.mm 형식으로 변환
- 이번달 → 현재 연월
- 지난달 → 직전 연월
- "4월" → 현재 연도 기준 2026.04
- "2026년 3월", "2026.3", "2026-03" → 2026.03
- 월 언급 없으면 추출하지 않음 (전체 기간 검색)
